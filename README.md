# HEX → RGB конвертер

React-приложение для конвертации цветов из формата HEX в RGB с валидацией.

## 🎯 Функциональность

- Конвертация HEX → RGB
- Валидация ввода (7 символов, включая `#`)
- Отображение ошибки при неверном формате
- Автоматическая смена фона на выбранный цвет

## 🛠 Технологии

- React + TypeScript
- Хуки (useState)
- CSS-стилизация

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
